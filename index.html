Textos ✍

Os textos em HTML são essenciais para inserir informações nas aplicações web, e possuem diferentes tags para caracterizar diferentes tipos de textos. Dentre essas tags, as principais são:

<p></p> 👉 Principal tag de texto, usada para escrever parágrafos.

<span></span> 👉 Usada normalmente para pequenas informações, como legendas. Também usada para fazer containers in-line.

<i></i> 👉 Usada para transformar o conteúdo em itálico.

<b></b> 👉 Usada para transformar o conteúdo em negrito.

<br/> 👉 Usada para a função de quebra de linha.

Imagens 🖼️

As imagens são usadas nos sites para representar elementos, paisagens e milhares de outras coisas. Podem ser usadas através da tag <img>.

Estilos para Links 🎀

Os links, através da tag <a>, no geral, são muito utilizados para inserção de hyperlinks à nossa página web, através de textos, imagens e botões. Sua principal função é destinar o usuário para algum lugar. No caso do menu deste site, a função da tag é lhe direcionar para as diferentes sessões. No CSS, é possível estilizar a decoração de um <a>:

text-decoration 👉 Define o estilo do sublinhado do <a>.

color 👉 Define a cor do <a>.

Também é possivel definir seu estilo em relação às suas quatro fases:

a:link 👉 Define o estilo do link no estado inicial.

a:visited 👉 Define o estilo do link visitado.

a:hover 👉 Define o estilo do link quando passa-se o mouse sobre ele.

a:active 👉 Define o estilo do link ativo (o que foi "clicado").

Responsividade 📱

A responsividade nas páginas web permite que o conteúdo se ajuste de maneira correta e proporcional aos diferentes tamanhos de tela, garantindo uma experiência consistente e agradável para os usuários, independentemente do dispositivo que estão usando.

Animações e Transições 🪐

As animações em CSS permitem que você adicione movimento e interatividade aos elementos de suas páginas da web. Com CSS, através dos KeyFrames, você tem acesso a uma variedade de propriedades que permitem criar animações sem a necessidade de JavaScript.

#box-animation {
animation: growing 1.5s alternate infinite ease-in ;
}

@keyframes growing {
   0% {
transform: scale(0.8) ;
background-color: #77dd77;
   }
    100% {
transform: scale(1.2);
background-color: #84b6f4;
   }
}

Consistência 🧱

A consistência é um fator fundamental na construção de qualquer página, e ela se traduz no uso constante e padronizado de tamanhos, cores, fontes e layouts em uma estrutura web, de modo a representar ao usuário uma estilização uniforme e homogênea do início ao fim.
